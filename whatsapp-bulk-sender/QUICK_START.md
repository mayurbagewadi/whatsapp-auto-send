# 🚀 Quick Start Guide - 3 Minutes to First Send

---

## Step 1: Add Icons (2 minutes)

**Easiest method:**

1. Visit: https://favicon.io/favicon-generator/
2. Enter text: `WA`
3. Change background color: `#25D366`
4. Click "Download"
5. Extract the ZIP file
6. Copy these files to the `icons/` folder:
   - `favicon-16x16.png` → rename to `icon16.png`
   - `favicon-32x32.png` → rename to `icon48.png`
   - `android-chrome-192x192.png` → rename to `icon128.png`

**Or:** Use any 3 PNG images (different sizes) and name them `icon16.png`, `icon48.png`, `icon128.png`

---

## Step 2: Load Extension (30 seconds)

1. Open Chrome
2. Go to: `chrome://extensions/`
3. Enable **"Developer mode"** (toggle in top-right corner)
4. Click **"Load unpacked"** button
5. Select the `whatsapp-bulk-sender` folder
6. Extension loads! Icon appears in toolbar

---

## Step 3: Test Send (30 seconds)

1. **Open WhatsApp Web:** https://web.whatsapp.com
2. **Log in** (scan QR code if needed)
3. **Click extension icon** in Chrome toolbar
4. **Wait for green dot:** "WhatsApp Connected"
5. **Paste your own number first** (for testing):
   ```
   919876543210
   ```
   *(Replace with your number including country code)*
6. **Type test message:**
   ```
   This is a test from my bulk sender
   ```
7. **Click "Start Sending"**

✅ You should receive the message on your phone!

---

## Step 4: Real Bulk Send

Once test works, paste multiple numbers:

```
919876543210
14155552671
447911123456
```

**Format rules:**
- ✅ Country code first (91, 1, 44, etc.)
- ✅ No spaces
- ✅ No + symbol
- ✅ One number per line

**Safe settings:**
- Delay: **10 seconds** (default, safe)
- Randomize: **ON** (checked)

Then click **"Start Sending"** and watch the progress!

---

## Troubleshooting (if needed)

**Red dot "WhatsApp Web not open"**
→ Open https://web.whatsapp.com first

**Yellow dot "Please scan QR code"**
→ You're not logged in to WhatsApp Web

**Button stays grayed out**
→ Wait 30 seconds, or refresh WhatsApp Web tab

**Message shows "Failed"**
→ Check number format (must include country code)

---

## Tips for Success

✅ **Keep WhatsApp tab open** while sending
✅ **Don't close Chrome** during sending
✅ **Use 10+ second delay** to avoid bans
✅ **Test with your own number first**
✅ **Start small** (5-10 numbers first time)
✅ **Randomize is your friend** (keeps it human-like)
